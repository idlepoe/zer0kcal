# 🧊 제로칼로리 (Zer0kcal)

> "이건... 제로칼로리야!"  
> 🍰 죄책감 없이 먹는 즐거움, 감성 위로 앱 제로칼로리

**제로칼로리 (Zer0kcal)** 는  
음식 사진을 AI가 분석하고 감탄사를 더해  
"이건 제로칼로리야!" 라고 위로해주는 감성 음식 공유 플랫폼입니다.
